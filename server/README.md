# Ridiculus server

This is a local server using mongodb. Just run npm start in the /server folder (after installing mongodb). It uses port 3000.

## Queries 

POST : IP/postes/

PUT : IP/postes/:id

DELETE : IP/postes/:id

FIND : IP/postes/      Get all

FIND : IP/postes/:id   Get one


Same for users (excepted PUT) or commentaires.
